## 🌟 プロジェクトについて

**こんとりのしま**へようこそ！ここはあなたの GitHub コントリビューションが美しい 3D 島として生まれ変わる不思議な場所です。

毎日のコミット、プルリクエスト、イシューの作成など、あなたの GitHub での活動が、どうぶつの森風の素敵な島を作り出します。コントリビューションが増えるほど、島はどんどん大きく、そして美しく成長していきます！

## 🎮 コンセプト

> 「あなたのコントリビューションが、あなただけの島を作り出す」

プログラミングの世界で頑張るあなたに、素敵なご褒美を。毎日のコントリビューションが、目に見える形で島の成長として表現されます。

- 🌱 コントリビューションが増えると島が成長
- 🏠 活動の種類によって異なる建物が出現
- 🌈 長期的な活動で島の風景が変化
- 🎨 自分だけの島をカスタマイズ可能

## 🚀 環境構築

1. リポジトリのクローン

```bash
git clone https://github.com/Ojoxux/Contri_Island.git
cd Contri_Island.git
```

2. 依存パッケージのインストール

```bash
npm install
```

3. 環境変数の設定

```bash
cp .env.example .env
# .envファイルを編集して必要な環境変数を設定
```

4. 開発サーバーの起動

```bash
npm run dev
```

5. ブラウザで開く

```
http://localhost:3000
```

## 🌈 機能

- 🏝️ 3D インタラクティブな島の表示
- 📊 コントリビューション統計
- 🎯 週間・月間の目標設定
- 🎨 島のカスタマイズ
- 📱 レスポンシブデザイン
